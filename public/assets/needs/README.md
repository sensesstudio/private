# "Needs" category photos

The Search → Needs cards load `assets/needs/<id>.jpg`. Drop a JPG here named by
the id below. Until a file exists, the card shows a soft gradient placeholder.

Recommended: landscape ~1200×800, optimised < 300 KB (the app also compresses).

| File name        | Category |
|------------------|----------|
| `reformer.jpg`   | Rehab |
| `contemporary.jpg` | Core & Strength |
| `prenatal.jpg`   | Pre / Postnatal & Restore |
| `mat.jpg`        | Mobility |
| `foundations.jpg`| Posture & Alignment |
| `gyrotonic.jpg`  | 1:1 The GYROTONIC® |
