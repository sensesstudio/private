# Teacher portraits

here named by the teacher id below. Until a file exists, that teacher shows their
initials placeholder (no broken image). Used everywhere — client, teacher and
admin portals — via the shared Avatar.

Recommended: square-ish portrait, ~800×800, face centred, optimised < 300 KB.
